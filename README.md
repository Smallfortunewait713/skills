## 🖥️ What this is

skills is a set of Angular helper files for coding agents. It helps with common Angular work, like building components, setting up a new app, and following current Angular practices.

It is made for people who want better Angular code and a cleaner project setup. The tools focus on clear structure, modern Angular features, and simple app management.

## ✨ What you can use it for

- Build Angular components
- Set up services and data flow
- Work with forms
- Use routing for page changes
- Set up server-side rendering
- Add accessibility labels and ARIA hints
- Handle animations
- Apply styles with a clear structure
- Prepare tests
- Create new Angular apps with the CLI
